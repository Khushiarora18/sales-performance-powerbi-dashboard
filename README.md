# sales-performance-powerbi-dashboard
Interactive Power BI dashboard analyzing retail sales performance using Power Query and DAX.

## Project Overview
This project presents an interactive Power BI dashboard designed to analyze retail sales performance across regions, product categories, and time periods. The dashboard enables stakeholders to monitor key performance indicators and identify business insights for strategic decision-making.

## Objectives
- Analyze monthly and quarterly sales performance
- Identify top-performing product categories
- Evaluate regional sales distribution
- Monitor profitability through key metrics

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Excel

## Data Preparation
Raw sales data was cleaned and transformed using Power Query. A relational data model was built to establish connections between tables and ensure efficient data analysis.

## Data Model
The data model consists of relationships between sales, product, and regional datasets to support dynamic analysis.

## Dashboard Features
- KPI cards displaying Total Sales, Profit Margin, and Category Performance
- Interactive filters for time, region, and product categories
- Trend visualizations for monthly and quarterly sales
- Category-level performance insights

## Key Insights
- Certain regions contribute significantly higher revenue than others.
- Seasonal demand patterns were identified through monthly sales trends.
- Specific product categories generate the majority of profits.
